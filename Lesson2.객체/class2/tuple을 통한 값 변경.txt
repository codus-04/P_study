* tuple을 이용하면, 다음과 같이 학생 정보를 표현할 수 있다.
student1 = (90,80,90)
kor, eng, math = student1
print(kor,eng,math) // 90 80 90

- tuple에서는 class와 같이 기본값이라는 개념이 없다. 아직 원하지 않은 값이 정해지지 않은 경우라면,
 다음과 같이 적절한 초기값을 직접 넣어주며 생성해야 한다.

student1 = (0,0,0)
kor,eng,math = student1
print(kor, eng, math) // 0 0 0

- 그렇다면 tuple에 이미 들어있는 값은 어떻게 바꿀 수 있을까?
tuple은 immutable하기 때문에 원소 값을 바꿀 수 없다고 했었다.
따라서 tuple에 있는 원소의 값을 의미상 바꾸기 위해서는 다음과 같이 꼭 새로운 tuple을 생성해줘야 한다.
다음은 수학 점수만 90으로 바꾼 예이다.
student1 = (0,0,0)
kor, eng, math = student1
print(kor, eng, math) // 0 0 0

student1 = (kor, eng, 90)

kor, eng, math = student1
print(kor, eng, math) // 0 0 90

* Side Note
tuple 이용시 만약 수학 점수만 필요한 경우라면, 코드를 어떻게 작성하는게 좋을까?
바로 Unpacking시 국어, 영어 점수와 같이 필요없는 곳에는 _ 를 넣어 사용하지 않겠다는 의미부여를 해주는 것이 좋다.
student1 = (90,80,90)
_,_,math = student1 // 수학 점수만 사용하겠다는 의지
print(math)
