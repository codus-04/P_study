// Side Note2

어떻게 class 대신 tuple을 이용해 원하는 형태로 담을 수 있나?
student1 = (90,80,90)

다만, tuple을 이용하는 경우에는 각 값의 의미하는것이 무엇인지 알기 어렵기 때문에, 값을 이용할 때 되도록이면
Unpacking을 통해 꼭 적절한 이름을 붙여준 뒤 사용하는 것을 권장한다.

다음은 Unpacking을 이용하지 않고 국어, 영어, 수학 점수 값을 출력한 예이다.
tuple에서 값을 참조할 때에는 다음과 같이 index기반으로 참조가 가능하다.
다만 이 방법은 가독성이 매우 떨어져 권장되지 않는다.
student1 = (90,80,90)
print(student1[0],student1[1],student1[2]) // 90 80 90

따라서 다음과 같이 각 원소에 대해 Unpacking시 적절한 변수 이름을 사용하여 의미를 부여한 뒤, 사용해라.
student1 =(90,80,90)
kor,eng,math = student1
print(kor,eng,math) // 90 80 90
