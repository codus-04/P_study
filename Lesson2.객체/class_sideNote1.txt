* tuple
python3 에서 새로운 형태를 정의할 때 class를 만들지 않고 진행하는 방법도 있다.
크게 복잡하지 않은 문제의 경우에는 class를 만들지 않고 tuple을 이용하여 문제를 해결하는 경우도 많다
def hello():
        return1,2
print(hello()) //(1,2)

위 코드의 결과에서처럼 소괄호로 둘러쌓인 형태로 정의되어 있는 것이 바로 tuple이다.
tuple은 리스트와 비슷하지만, 크게 2가지 차이점이 존재한다.
- 튜플에 들어가 있는 값은 수정할 수 없다.
- 튜플에 추가 원소를 넣거나 기존 원소를 제거하는 것은 불가하다.

즉, tuple은 immutable하다
튜플의 주된 역할은 패킹과 언패킹이 있다.
def hello():
    return 1,2

a,b = hello()
print(a,b,a+b)

이 경우 123 이라는 결과가 출력된다.
즉, a ,b =(1,2)를 하게 되면 a는 1, b는 2라는 값을 받게 되는 것이다.
우리는 이것을 튜플 언패킹(Tuple Unpacking)이라고 부른다.

이것을 활용하면, 하나의 함수로 여러개의 반환값을 갖도록 할 수 있고, 그것을 각각의 변수에 할당할 수 있다.

우리는 일반적으로 하나의 변수에 값을 여러개 대입할 때,
dt = [1,2,3,4,5]

이런식으로 리스트에 담아서 변수의 값을 담았었다.
dt = 1,2,3,4,5

이런식으로 리스트를 활용하지 않고 여러개의 값을 담으려고 한다면, 튜플 형식으로 묶여서 값을 집어 넣게 된다.

위에서 다뤘던 언패킹과 반대방향 연산으로, 패킹이라고 할 수 있다.
